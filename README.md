# Elder Scrolls Legends Cards Browser

## Local Run

1. [Install Yarn](https://classic.yarnpkg.com/en/docs/install)

2. Install dependencies 
    ```
    yarn install
    ```
3. Start the application
    ```
    yarn start
    ```
4. Navigate to [http://localhost:3000](http://localhost:3000)
